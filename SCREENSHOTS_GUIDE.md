# Chrome Web Store Screenshots Guide

## Requirements

Chrome Web Store requires 3-5 promotional screenshots (1280x800 or 640x400 pixels recommended).

**Format:** PNG, JPG, or GIF
**Recommended:** 1280x800 pixels or 640x400 pixels

## Screenshot Recommendations

### Screenshot 1: Hero Shot - Main Feature
**Goal:** Show the core value proposition
- Capture a screenshot being annotated with multiple tools
- Show annotations toolbar visible at the top
- Include text, arrows, and shapes
- Make it look professional and clean
**Title on image:** "Annotate with precision"

### Screenshot 2: Gallery/Recents View
**Goal:** Demonstrate easy management
- Show the popup with the "Recents" gallery
- Display 2-3 thumbnail previews
- Show the "View Recents" button and Downloads button
**Title on image:** "Your annotations, organized"

### Screenshot 3: Text Annotation
**Goal:** Highlight text tool functionality
- Capture an annotated screenshot with multiple text boxes
- Show different font sizes and colors
- Show the color picker in use
**Title on image:** "Add clear, professional labels"

### Screenshot 4: Blur/Redaction Feature
**Goal:** Emphasize privacy-focused redaction
- Show a screenshot with sensitive areas blurred
- Display the blur selection outline
- Include the "Invert ON" button and Apply/Cancel buttons
**Title on image:** "Redact sensitive information"

### Screenshot 5: Drawing Tools
**Goal:** Showcase diverse annotation capabilities
- Show a technical diagram or screenshot
- Annotate it with arrows pointing to issues
- Use rectangles and circles to highlight areas
- Include multiple drawing tools in action
**Title on image:** "Express ideas with drawings"

## Tips for Better Screenshots

1. **Use Test Images** - Create screenshots on simple, recognizable content
2. **Show UI Clearly** - Make sure the toolbar and buttons are visible and readable
3. **Add Watermarks** - Consider adding "Highlark" watermark to screenshots
4. **Use Contrast** - Ensure text is readable (dark text on light background preferred)
5. **Aspect Ratio** - Use consistent aspect ratios across all screenshots
6. **Professional Look** - Use neutral backgrounds, avoid personal content
7. **Text Overlay** - Consider adding benefit-focused text overlays on images

## How to Create Screenshots

### Option 1: Manual Screenshot + Highlark
1. Take screenshot of content using Highlark
2. Annotate it using Highlark's tools
3. Download as PNG
4. Resize if needed (use image editor or online tool)

### Option 2: Using System Screenshot Tool
1. Use system screenshot tool (Mac: Cmd+Shift+4, Windows: Print Screen)
2. Open Highlark and capture it as the content
3. Annotate demonstration features
4. Export and prepare for store

## File Naming Convention

Save screenshots as:
- `screenshot-1-hero.png`
- `screenshot-2-gallery.png`
- `screenshot-3-text.png`
- `screenshot-4-blur.png`
- `screenshot-5-drawing.png`

## Where to Save

Once created, prepare screenshots for upload to Chrome Web Store:
- Store them in `assets/chrome-store-screenshots/` folder
- Include high-res versions for reference
- Keep PNG format for quality

## Uploading to Chrome Store

1. Go to Chrome Web Store Developer Console
2. Select your extension
3. Go to **Store Listing** tab
4. Scroll to **Promotional tiles** section
5. Upload up to 5 screenshots (at least 3 required)
6. Add brief, benefit-focused captions for each

### Caption Examples

1. "Annotate screenshots with text, arrows, shapes, and more"
2. "Keep all your annotations organized in the local gallery"
3. "Add professional text annotations with custom colors and sizes"
4. "Blur sensitive information to protect privacy"
5. "Use drawing tools to highlight and communicate issues"
